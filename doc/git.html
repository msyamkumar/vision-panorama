Git logs and credits

Source code repository link

https://github.com/msyamkumar/vision-panorama/

To run the program, provide the path where the required images are there along with image_list.txt file. The images should be provided in left shifted ordering. Then, just execute "panaroma.m" which would write the panroma to a file in the same directory.
